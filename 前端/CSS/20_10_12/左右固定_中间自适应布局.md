## float 浮动布局

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>浮动布局</title>
</head>

<style>
    html, body {
        height: 100%;
        margin: 0;
        padding: 0;
    }

    .left {
        width: 100px;
        height: 100%;
        float: left;
        background: pink;
    }

    .right {
        width: 100px;
        height: 100%;
        float: right;
        background: cadetblue;
    }

    .main {
        width: 100%;
        height: 100%;
        background: green;
    }
</style>

<body>
    <div class="left">left</div>
    <div class="right">right</div>
    <div class="main">main</div>
</body>

</html>
```

## position 布局

</body>

</html>

## 圣杯布局

父元素 + padding

```html
<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>

<style>
    html,
    body {
        margin: 0;
        padding: 0;
        height: 100%;
    }

    body {
        padding: 0 100px;
    }

    .main {
        width: 100%;
        height: 100%;
        float: left;
        background: blue;
    }

    .left {
        width: 100px;
        height: 100%;
        background: red;
        float: left;
        margin-left: -100%;
        position: relative;
        left: -100px;
    }

    .right {
        width: 100px;
        height: 100%;
        background: red;
        float: left;
        margin-left: -100px;
        position: relative;
        right: -100px;
    }
</style>
<body>
    <div class="main"></div>
    <div class="left"></div>
    <div class="right"></div>
</body>
</html>
```

## 双飞翼布局

子元素 + margin

```html
<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>

<style>
    html,
    body {
        height: 100%;
        margin: 0;
        padding: 0;
    }

    .main {
        width: 100%;
        height: 100%;
        float: left;
    }

    .middle {
        margin: 0 100px;
        height: 100%;
        background: blue;
    }

    .left {
        width: 100px;
        height: 100%;
        background: red;
        float: left;
        margin-left: -100%;
    }

    .right {
        width: 100px;
        height: 100%;
        background: red;
        float: left;
        margin-left: -100px;
    }
</style>

<body>
    <div class="main">
        <div class="middle"></div>
    </div>
    <div class="left"></div>
    <div class="right"></div>
</body>
</html>
```
